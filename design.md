# Дизайн-система платформы "Конфорка"

## Философия дизайна

### Визуальная концепция
Платформа "Конфорка" - это инновационное пространство, где наука встречается с предпринимательством. Дизайн должен отражать:
- **Научную точность** - чистые линии, структурированная информация, логичная иерархия
- **Инновационность** - современные визуальные решения, динамические элементы
- **Доступность** - интуитивный интерфейс для разных категорий пользователей
- **Профессионализм** - серьезный подход к научным исследованиям и бизнесу

### Цветовая палитра

#### Основные цвета
- **Primary Blue**: #2563EB - основной цвет бренда, надежность и инновации
- **Secondary Teal**: #0891B2 - дополнительный цвет, научная точность
- **Accent Orange**: #EA580C - акцентный цвет, энергия идей
- **Neutral Gray**: #374151 - нейтральный цвет для текста

#### Семантические цвета
- **Success Green**: #059669 - успех, завершение проектов
- **Warning Amber**: #D97706 - предупреждения, внимание
- **Error Red**: #DC2626 - ошибки, критические состояния
- **Info Cyan**: #0891B2 - информационные сообщения

#### Фоновые цвета
- **Background Light**: #F8FAFC - основной фон
- **Background Dark**: #1F2937 - темный фон для контраста
- **Surface White**: #FFFFFF - поверхности, карточки
- **Surface Gray**: #F3F4F6 - вторичные поверхности

### Типографика

#### Шрифты
- **Display Font**: "Inter" - современный геометрический шрифт для заголовков
- **Body Font**: "Source Sans Pro" - читаемый шрифт для основного текста
- **Code Font**: "JetBrains Mono" - моноширинный шрифт для кода и данных

- **H1**: 48px, font-weight: 700, line-height: 1.2

#### Основной текст
- **Body Large**: 18px, font-weight: 400, line-height: 1.6

### Компоненты интерфейса

#### Кнопки
- **Primary Button**: синий фон, белый текст, закругленные углы 8px
- **Secondary Button**: прозрачный фон, синий бордер и текст
- **Icon Button**: круглая кнопка для иконок, размеры 32px, 40px, 48px
- **Floating Action Button**: оранжевая кнопка для основных действий

#### Карточки
- **Project Card**: белый фон, тень, закругленные углы 12px
- **User Card**: компактная карточка пользователя с аватаром
- **Equipment Card**: карточка оборудования с изображением и характеристиками
- **Mentor Card**: карточка ментора с рейтингом и экспертизой

#### Формы
- **Input Field**: белый фон, серый бордер, фокус - синий бордер
- **Textarea**: растягиваемое поле для длинного текста
- **Select**: выпадающий список с поиском
- **Checkbox/Radio**: кастомные стилизованные элементы

#### Навигация
- **Top Navigation**: фиксированная верхняя панель с логотипом и меню
- **Side Navigation**: боковая панель для разделов платформы
- **Breadcrumbs**: хлебные крошки для навигации по разделам
- **Tabs**: вкладки для переключения между подразделами

### Иконография

#### Стиль иконок
- **Line Icons**: линейные иконки толщиной 2px
- **Filled Icons**: залитые иконки для активных состояний
- **Size Variants**: 16px, 20px, 24px, 32px, 48px

#### Категории иконок
- **Навигация**: домой, поиск, профиль, настройки
- **Действия**: добавить, редактировать, удалить, сохранить
- **Контент**: проекты, оборудование, менторы, R&D заказы
- **Статус**: успех, ошибка, предупреждение, информация

### Анимации и переходы

#### Принципы анимации
- **Subtle Motion**: неброские, естественные движения
- **Purposeful**: анимация должна служить функциональной цели
- **Consistent**: единые тайминги и easing функции
- **Accessible**: уважать настройки предпочтений пользователя

#### Тайминги
- **Fast**: 150ms для быстрых интеракций (hover, focus)
- **Normal**: 300ms для стандартных переходов
- **Slow**: 500ms для сложных анимаций

#### Easing функции
- **Ease Out**: для появления элементов
- **Ease In**: для исчезновения элементов
- **Ease In Out**: для движения элементов

### Layout и Grid система

#### Grid система
- **Container**: максимальная ширина 1200px, центрирование
- **Columns**: 12-колоночная система с отступами 24px
- **Breakpoints**: 
  - Mobile: 320px - 768px
  - Tablet: 768px - 1024px
  - Desktop: 1024px+

#### Отступы
- **Base Unit**: 8px для всех отступов
- **Small**: 8px (1 unit)
- **Medium**: 16px (2 units)
- **Large**: 24px (3 units)
- **XLarge**: 32px (4 units)

### Состояния компонентов

#### Hover состояния
- **Button**: изменение цвета фона на 10% темнее
- **Card**: появление тени увеличенной интенсивности
- **Link**: подчеркивание или изменение цвета

#### Focus состояния
- **Input**: синяя рамка толщиной 2px
- **Button**: синяя обводка для клавиатурной навигации

#### Loading состояния
- **Skeleton**: анимированные плейсхолдеры для контента
- **Spinner**: круговой индикатор загрузки
- **Progress Bar**: линейный индикатор прогресса

### Доступность

#### Цветовой контраст
- **Text on Background**: минимальный контраст 4.5:1
- **Large Text**: минимальный контраст 3:1
- **Interactive Elements**: четкое визуальное разграничение

#### Навигация

### Компоненты для специфических разделов

#### Маркетплейс проектов
- **Project Grid**: сетка карточек проектов с фильтрами
- **Project Detail View**: детальная страница проекта с описанием
- **Team Builder**: интерфейс для формирования команды

#### Маркетплейс оборудования
- **Equipment Catalog**: каталог оборудования с фильтрами
- **Booking Calendar**: календарь бронирования
- **Equipment Detail**: страница с характеристиками оборудования

#### Система менторства
- **Mentor Directory**: каталог менторов с фильтрами
- **Consultation Booking**: система бронирования консультаций
- **Video Chat Interface**: интерфейс для видеозвонков

#### R&D заказы
- **Order Board**: доска с активными заказами
- **Submission Interface**: интерфейс для подачи решений
- **Evaluation Dashboard**: панель для оценки решений

### Микровзаимодействия

#### Уведомления
- **Toast Notifications**: всплывающие уведомления в правом верхнем углу
- **In-app Notifications**: счетчики и индикаторы в интерфейсе
- **Email Templates**: шаблоны email уведомлений

#### Обратная связь
- **Success States**: зеленые индикаторы успешных действий
- **Error Handling**: красные сообщения об ошибках с пояснениями
- **Loading States**: индикаторы загрузки для длительных операций

### Брендинг

#### Логотип
- **Primary Logo**: полный логотип с названием
- **Icon Logo**: компактная иконка для маленьких размеров
- **Wordmark**: только текстовое название

#### Брендовые элементы
- **Color Accent**: оранжевый цвет для выделения ключевых элементов
- **Geometric Shapes**: абстрактные геометрические формы в дизайне
- **Scientific Imagery**: элементы, напоминающие научные исследования

Эта дизайн-система обеспечивает единообразие и профессиональный вид всей платформы, подчеркивая её инновационную и научную направленность.
