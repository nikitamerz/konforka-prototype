# Платформа "Конфорка" - Суперприложение для развития науки

## Описание проекта

"Конфорка" - это инновационная платформа, объединяющая четыре маркетплейса и социальную сеть для развития науки и технологического предпринимательства. Проект решает проблему доступности оборудования для стартапов и создает мост между академической наукой и реальным сектором экономики.

## Функциональные возможности

### 1. Маркетплейс идей и проектов
- Публикация идей для исследований и стартапов
- Формирование команд из инженеров, ученых и менеджеров
- Управление проектами и задачами
- Система рейтингов и отзывов

### 2. Маркетплейс оборудования
- Аренда научного оборудования от вузов и предприятий
- Календарь доступности и онлайн-бронирование
- Система ценообразования и оплаты
- Управление доступом и сертификацией

### 3. Маркетплейс менторства
- Консультации с экспертами и предпринимателями
- Видеосвязь и онлайн-встречи
- Система рейтингов и отзывов
- База знаний и советов

### 4. Маркетплейс R&D-заказов
- Размещение заказов от промышленных предприятий
- Конкурсы с призовыми фондами
- Система оценивания и выбора победителей
- Юридическое сопровождение сделок

## Техническая архитектура

### Frontend
- **HTML5** с семантической разметкой
- **Tailwind CSS** для стилизации
- **JavaScript (ES6+)** для интерактивности
- **Plotly.js** для визуализации данных
- **Font Awesome** для иконок
- **Google Fonts** (Inter, Source Sans Pro) для типографики

### Backend (запланировано)
- **FastAPI** для REST API
- **PostgreSQL** для основной базы данных
- **SQLAlchemy** для ORM
- **JWT** для аутентификации
- **Redis** для кеширования

### База данных
- **Пользователи** (ученые, менторы, поставщики оборудования)
- **Проекты** (идеи, команды, задачи)
- **Оборудование** (каталог, бронирования, календарь)
- **Менторство** (консультации, отзывы, расписание)
- **R&D заказы** (конкурсы, заявки, оценивание)

## Установка и запуск

### Локальный запуск
1. Склонируйте репозиторий
2. Откройте `index.html` в браузере
3. Для полной функциональности запустите локальный сервер:
   ```bash
   python -m http.server 8000
   ```
4. Откройте http://localhost:8000

### Развертывание
1. Загрузите все файлы на веб-сервер
2. Обеспечьте доступ к статическим файлам
3. Настройте маршрутизацию для SPA (если требуется)
